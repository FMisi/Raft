- A projektet a NetBeans 8.2 IDE segitsegevel keszitettem el

- Java letöltése
   - A Java sikeres telepítése a /Raft/dist/ mappában lévő .jar fájlra való dupla kattintással ellenőrizhető
   - A Java letölthető többek között az alábbi weboldal segítségével ezen README fájl megírásának pillanatában
   - https://okt.sed.hu/prog1/gyakorlat/telepites/

- NetBeans 8.2 projekt IDEA-ba importálása
   - Ez egyszerű, csak hozzunk létre új projektet IDEA-ban, majd a /Raft/ mappa tartalmát
     másoljuk át az új projekt mappájába (nem baj, ha az /src/ felülíródik) és már működik is!:D

- Irányítás
   - Az irányítási útmutató a raftiranyitas.jpg fájl megnyitásával elérhető

- Operációs rendszerek, amiken a projekt tesztelve lett
   - Windows 10 Home 64 bit
   - Windows 7 Ultimate 64 bit
   - Kali Linux 2021.1

- Ismert bug
   - Az Ismert bug vázlatpont az általam ismert egyetlen bug-ot ismerteti
   - Ezt a bug-ot ezúton kérlek, hagyd figyelmen kívül az értékelés során
	- A hálóból ha több van, ugyanarra a mezőre többet is le tudunk helyezni,
	  azonban 12-nél nem tudunk többet lehelyezni (mivel annyi mező van összesen
	  a ki nem bővített tutaj körül)

- Javítási segédlet
   - Ezen Javítási segédlet a követelmények menti értékeléshez való segítségnyújtást valósítja meg
      - A pálya tetején nem minden cselekvéskor úkeletkezik új nyersanyag, hanem összesen 3 nyersanyag keletkezik
        a cselekvések számának függvényében
      - A kibővítés nem működik
      - A cápa valóban nem ütközik a tutajjal és nem megy le a térképről, de körkörösen mozog, és nem követi a játékost
      - Az előző vázlatpont miatt lándzsát sem tudunk használni, csak craft-olni
      - Mentés és betöltés nincs
      - A fenti vázlatpontok összesen 25%-ot érnek, reményemet fejezem ki ezúton, hogy 5%-nál nem nagyobb azon
	szempontok száma amelyek elkerülték körültekintő figyelmemet
      - Csak halat tudunk sütni, burgonyát nem, de a feladatban "legalább 1"-ként volt megadva azon ételek száma
        amelyekre a sütés megvalósítása elengedhetetlenül szükséges volt

- Remélem, tetszeni fog! Jó játékot!
